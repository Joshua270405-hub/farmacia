# FarmaPlus RD - Formato tipo Hacienda Espinal

Estructura:
- index.html
- styles.css
- script.js
- assets/  (todas las imágenes dentro)

Para subir a GitHub:
1. Descomprime este ZIP.
2. Sube los archivos:
   - index.html
   - styles.css
   - script.js
   - carpeta assets completa
3. Luego ve a Settings > Pages.
4. Selecciona Deploy from a branch > main > /root.
5. Save.

IMPORTANTE:
No cambies el nombre de la carpeta assets.
No subas solo el ZIP; descomprímelo primero.
